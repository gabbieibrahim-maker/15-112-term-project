# 15-112-term-project

Color Switch-Inspired 2D Game | Python (15-112 Term Project)

# April 2026

An endless 2D mobile-style game built in Python using object-oriented architecture and the CMU Graphics library.

# How to Play

Press the Space Bar to jump and start the game.

Navigate the menus using your mouse.

Avoid hitting the bottom of the screen or touching the wrong colors!

# Grading & Debugging Shortcuts

Press 'G' to access the grading menu to test specific isolated obstacles.

Press 'S' or click on "Skins" button to open the skins page to customize your character.

Press 'M' or click on "Return to main menu" button to return to the main menu.

May need to press twice on center of skin (pink circle) when customizing your character.
